# skibtracker ⚽

Live World Cup 2026 semi-final & final goal and key event tracker.

Pulls real-time data from worldcup26.ir — no API key needed.

## Tracks
- Both semi-finals (July 14 & 15)
- The final (July 19)

## Events reported
- Goals (open play, own goals, penalties)
- Red cards
- Live score + match minute

## Automations
Powered by Superagent — sends WhatsApp alerts on goals and key events during live matches.

## Script
`wc_tracker.py` — run directly or triggered via automation.
